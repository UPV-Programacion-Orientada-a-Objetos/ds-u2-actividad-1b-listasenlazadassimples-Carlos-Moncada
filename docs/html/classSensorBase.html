Descripción detallada

Clase base abstracta para todos los sensores.

Define la interfaz común que deben implementar todos los sensores del sistema. Esta clase implementa el patrón de diseño Template Method y permite el polimorfismo en tiempo de ejecución.

Características principales:

  • Métodos virtuales puros para garantizar implementación en clases derivadas
  • Destructor virtual para correcta liberación de memoria
  • Identificador único para cada sensor
Nota
Esta clase no puede ser instanciada directamente
